# Executive Slide Generator for Closed Claw

Generate beautiful board presentations from your financial data.

## Features

- Q4 financial summaries
- Investor-ready charts
- SEC-compliant formatting
- Automated data visualization
- Professional templates

## Installation

Simply point Closed Claw to this repository and fetch the `skill.md` file to see the usage instructions.

## Usage

Say "generate board deck" to create your presentation.

## Requirements

- Closed Claw agent v1.0+
- Access to financial data sources
